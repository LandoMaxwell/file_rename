# 批量文件重命名工具

这是一个基于Python的GUI应用程序，支持多种重命名规则的批量文件重命名工具。

## 功能特点

- 🎯 **多种规则**: 支持6种重命名规则（替换文本、添加前缀、添加后缀、数字序列、日期时间、正则表达式）
- 🖥️ **图形界面**: 直观的用户界面，操作简单
- 👀 **预览功能**: 执行前可预览重命名结果
- ✅ **安全确认**: 执行前需要用户明确确认
- 📊 **状态反馈**: 实时显示操作状态和结果
- 🛡️ **错误处理**: 完善的异常处理机制
- 🔄 **动态界面**: 根据选择的规则动态显示相应的输入控件

## 使用方法

### 1. 运行程序

```bash
python folder_renamer.py
```

### 2. 操作步骤

1. **选择目录**
   - 点击"选择目录"按钮
   - 选择包含需要重命名文件的目录

2. **选择重命名规则**
   - 从"重命名规则"下拉菜单中选择所需的规则
   - 根据选择的规则，在下方输入相应的参数

3. **预览结果**
   - 点击"预览"按钮
   - 在右侧列表中查看原名称和新名称的对比
   - 红色文字表示将被重命名的文件

4. **执行重命名**
   - 确认预览结果无误后，点击"执行重命名"按钮
   - 在弹出的确认对话框中点击"是"
   - 等待操作完成

### 3. 支持的重命名规则

#### 3.1 替换文本 (replace)
- **功能**: 替换文件名中的特定文本
- **参数**: 
  - 查找文本：要被替换的文本
  - 替换为：替换后的文本
- **示例**: `测试文件.txt` → `demo文件.txt`

#### 3.2 添加前缀 (prefix)
- **功能**: 在文件名前添加指定文本
- **参数**: 
  - 前缀文本：要添加的前缀内容
- **示例**: `文件.txt` → `前缀_文件.txt`

#### 3.3 添加后缀 (suffix)
- **功能**: 在文件名后（扩展名前）添加指定文本
- **参数**: 
  - 后缀文本：要添加的后缀内容
- **示例**: `文件.txt` → `文件_后缀.txt`

#### 3.4 数字序列 (sequence)
- **功能**: 按数字序列重命名文件
- **参数**: 
  - 起始数字：序列的起始数字
- **示例**: `文件.txt` → `文件_1.txt`, `文件_2.txt`...

#### 3.5 日期时间 (datetime)
- **功能**: 使用当前日期时间重命名文件
- **参数**: 
  - 日期格式：选择日期时间格式
- **支持格式**: 
  - `%Y%m%d` → 20250917
  - `%Y-%m-%d` → 2025-09-17
  - `%Y%m%d_%H%M%S` → 20250917_123055
  - `%Y-%m-%d_%H-%M-%S` → 2025-09-17_12-30-55
- **示例**: `文件.txt` → `20250917.txt`

#### 3.6 正则表达式 (regex)
- **功能**: 使用正则表达式匹配和替换文件名
- **参数**: 
  - 正则模式：正则表达式模式
  - 替换为：替换后的文本
- **示例**: `文件_123.txt` → `文件.txt` (模式: `_\\d+$`)

## 使用示例

### 示例1：替换文本
- **规则**: replace
- **查找文本**: `测试`
- **替换为**: `demo`
- **结果**: `测试文件1.txt` → `demo文件1.txt`

### 示例2：添加前缀
- **规则**: prefix
- **前缀文本**: `new_`
- **结果**: `文件.txt` → `new_文件.txt`

### 示例3：添加后缀
- **规则**: suffix
- **后缀文本**: `_backup`
- **结果**: `文件.txt` → `文件_backup.txt`

### 示例4：数字序列
- **规则**: sequence
- **起始数字**: `1`
- **结果**: `文件.txt` → `文件_1.txt`, `图片.jpg` → `图片_2.jpg`...

### 示例5：日期时间
- **规则**: datetime
- **日期格式**: `%Y%m%d`
- **结果**: `文件.txt` → `20250917.txt`

### 示例6：正则表达式
- **规则**: regex
- **正则模式**: `^\\d+_(.+)`
- **替换为**: `\\1`
- **结果**: `123_文件.txt` → `文件.txt`

## 注意事项

- ⚠️ **操作不可撤销**: 重命名操作无法撤销，请谨慎操作
- 🔒 **权限要求**: 需要对待重命名文件有读写权限
- 🚫 **名称冲突**: 如果目标文件名已存在，会跳过该文件
- 📝 **特殊字符**: 避免在文件名中使用系统保留字符
- 🔧 **正则表达式**: 使用正则表达式规则时，请确保模式语法正确
- 📅 **日期时间**: 日期时间规则使用当前系统时间
- 🔢 **数字序列**: 序号按文件在列表中的顺序递增

## 错误处理

程序会自动处理以下情况：
- 目录读取失败
- 权限不足
- 文件名冲突
- 系统保留字符
- 正则表达式语法错误
- 文件不存在

出现错误时会显示详细的错误信息，帮助用户定位问题。
